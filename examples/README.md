## Headless UI (实验性)

Headless UI 源码：使用 Zeus 构建的 Web Components 组件库（Button、Switch、Checkbox、Tabs、Dialog 等）。

| Example            | Description                                    |
| ------------------ | ---------------------------------------------- |
| `headless`         | `@zeus-ui/headless` 源码                       |
| `headless-demo`    | 在 vanilla JS 中消费 headless WC               |
| `use-headless-cli` | Zeus CLI scaffold 示例（`zeus-ui add button`） |

### React 生态

| Example              | Description                                  |
| -------------------- | -------------------------------------------- |
| `registry-react`     | React + Tailwind + headless 组件             |
| `use-headless-react` | React 19 完整展示所有 headless 组件          |
| `react-wrapper`      | 在 React 中以 `zeus:react:*` 虚拟模块消费 WC |

### Vue 生态

| Example            | Description                              |
| ------------------ | ---------------------------------------- |
| `registry-vue`     | Vue 3 + Tailwind + headless 组件         |
| `use-headless-vue` | Vue 3 完整展示所有 headless 组件         |
| `vue-wrapper`      | 在 Vue 中以 `zeus:vue:*` 虚拟模块消费 WC |
