## P1-3：Vite 插件的 external 合并需要留意

当前 Vite adapter 会从 component plugins 收集 external，然后返回：

```ts
build: {
  rollupOptions: {
    external: externals.length ? externals : undefined,
  },
}
```

这在多数场景能工作，但要确认不会覆盖用户自己配置的 `build.rollupOptions.external`。如果 Vite merge 行为没有按预期合并数组/函数，可能影响组件库用户手写 external。

建议后续改成更保险的逻辑：

```ts
config(config) {
  return mergeConfig(config, {
    build: {
      rollupOptions: {
        external: mergeExternal(config.build?.rollupOptions?.external, pluginExternals),
      },
    },
  })
}
```

或者在文档中明确：插件只会自动 external `react/vue`，用户仍可手动配置。
