# 已经符合预期的部分

## 1. transform 作用域已修正

现在 `rollup.ts` 不再全局编译所有 `.tsx/.jsx`，而是：

```ts
let shouldTransform = (_id: string) => false
```

随后在 `buildStart()` 里基于组件扫描规则创建 filter：

```ts
shouldTransform = createComponentTransformFilter({
  root,
  include,
  exclude,
})
```

transform 阶段也只在 `shouldTransform(id)` 为 true 时调用 `transformZeus()`。
