# 仍建议合并前修的 2 个问题

## P0-1：`jsx-runtime` / `jsx-dev-runtime` 的 `types` 指向错了

现在 `packages/zeus/package.json` 里是：

```json
"./jsx-runtime": {
  "types": "./jsx-runtime.js",
  "default": "./jsx-runtime.js"
},
"./jsx-dev-runtime": {
  "types": "./jsx-dev-runtime.js",
  "default": "./jsx-dev-runtime.js"
}
```

这里 `types` 指向了 `.js` 文件，应该指向 `.d.ts`。仓库里已经有 `jsx-runtime.d.ts` 和 `jsx-dev-runtime.d.ts`，内容也存在。

建议改成：

```json
"./jsx-runtime": {
  "types": "./jsx-runtime.d.ts",
  "default": "./jsx-runtime.js"
},
"./jsx-dev-runtime": {
  "types": "./jsx-dev-runtime.d.ts",
  "default": "./jsx-dev-runtime.js"
}
```

这个问题很小，但属于包出口错误，建议合并前修。
