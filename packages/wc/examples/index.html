🚀 Zeus Functional Web Components

✨ 核心特性

🎯 函数式编程

使用函数定义组件，而不是类。更简洁、更直观的代码结构。

🪝 Hooks 支持

类似 React Hooks 的 API，支持 useState、useEffect、useComputed 等。

⚡ 响应式状态

基于 signals 的细粒度响应式系统，自动追踪依赖关系。

🎨 声明式渲染

使用 JSX 语法，支持条件渲染、列表渲染等现代特性。

🔧 原生 Web Components

完全基于 Web Components 标准，支持 Shadow DOM 和 Light DOM。

📦 轻量级

极小的运行时体积，高性能，无虚拟 DOM 开销。

📝 基本用法

// 1. 导入必要的函数和 Hooks import { defineFunctionalWC, useState, useEffect, jsx } from '@zeus-js/wc' // 2. 定义函数式组件 function Counter(props) { const count = useState(0) useEffect(() => { console.log(`Count changed to: ${count()}`) }) return jsx('div', null, jsx('h2', null, 'Count: ', count()), jsx('button', { onClick: () => count(count() + 1) }, 'Increment') ) } // 3. 注册为 Web Component defineFunctionalWC('my-counter', Counter, { shadow: true, observedAttributes: ['initial-value'] })

🪝 可用的 Hooks

// 状态管理 const count = useState(0) const user = useState(null) // 计算属性 const doubleCount = useComputed(() => count() * 2) const userDisplayName = useComputed(() => user()?.name || 'Guest') // 副作用 useEffect(() => { console.log('Component mounted') return () => console.log('Component unmounted') }, []) // 引用 const inputRef = useRef(null) // 回调 const handleClick = useCallback(() => { console.log('Button clicked') }, []) // 记忆化 const expensiveValue = useMemo(() => { return heavyCalculation() }, [dependency])

🔧 配置选项

defineFunctionalWC('my-component', MyComponent, { shadow: true, // 是否使用 Shadow DOM styles: `...`, // 自定义样式 observedAttributes: ['prop1', 'prop2'], // 观察的属性 tagName: 'my-component' // 组件标签名 })
